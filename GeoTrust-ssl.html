WHAT IS AN SSL?

SSL stands for Secure Sockets Layer. This is the global security standard which creates an encrypted connection between web servers and browsers. The connection keeps all the data between the web servers and browsers secure and safe.

With an SSL, your privacy and sensitive information is yours. SSL certificates are considered as the business standard, used by millions of websites to protect their online transactions with customers.

HOW IT WORKS?

SSL is the solution for the security of e-commerce transactions and the rise of website reliability to customers with 3 steps as below.


An SSL encrypts sensitive information in online transactions

Each SSL is created for only one website

A reliable agency will verify website owners before issuing SSL certificates

WEBSITE AUTHENTICATION

When a computer connects with an authenticated website


  • First, the browser requires the website to provide authentication information.
  • Second, the website sends its SSL certificate to the browser.
  • Third, the browser verifies the SSL authentication, and then confirms with the website if the SSL is approved.
  • Then, the website sends back the digital signature for the encryption and decryption of transactions.
  • Finally, the data between the browser and website is encrypted.

SSL VERIFICATION

How a browser verifies an SSL?

Whenever a website sends its SSL certificate to a browser, the browser immediately sends this certificate to the servers containing digital approved certificates. Those servers are established by branded companies like GlobalSign and VeriSign.

An SSL technically applies the public encryption which the website and browser agree with each other about the keys for the next transactions. The keys will be changed for every transaction later, so hackers can’t encrypt them even if they have data from the server storing the certificate.
